# Language Locales Query Resource

Last Modified: 2021-10-26 | Code: APIBELL

To see the available Languages use the "/APIv2/Query/BusinessEssentials/LanguageLocales" dataset. The dataset can be executed without supplying values for the parameters.

### Shopmetrics CMS UI — Dataset Execution

![](assets/img_01.jpg)

### Postman

The content for the “post” parameter in the Body:

{"action":"exec","dataset":{"datasetname":"/Apps/SM/APIv2/Query/BusinessEssentials/LanguageLocales"},"parameters":[{"name":"SecurityObjectUserID","value":null},{"name":"MiscSettings","value":null}]}
